# rdScore 1.1.0 – Feature List

This document summarizes the main functional capabilities of rdScore.

---

## Core behaviour

- Single-window application
- Simple and predictable navigation
- Esc key always returns one level up
- No hidden state
- No automatic page memory

---

## PDF viewing

- Open PDF files
- Navigate pages
- Two-page reading mode
- Fast keyboard-based navigation

---

## Page extraction

Extraction dialog supports:

- single page
- page ranges

Examples:

7
10-14

Validation rule:

p1 ≤ p2

---

## Printing

Simple printing system:

- choose printer or PDF output
- print entire document
- print single page
- print page range

No print preview.

---

## Setlists

Setlists allow grouping several PDF scores together.

Capabilities:

- create setlists
- list existing setlists
- add files
- remove files
- reorder files

Navigation inside a setlist is **not forced to follow order**.

Users may open any entry directly.

---

## Setlist management

- preview before deletion
- confirmation required
- missing files automatically removed from setlist

---

## UI behaviour

- single window
- sidebar visible only in setlist mode
- sidebar toggle possible
- sidebar disappears when opening a PDF
